# Yolo Spam

![](header.png)

## Installation

OS X, Linux & Windows:

```sh
git clone https://github.com/donato-fiore/yolo-spam
```

## Setup

In yolo.py change ```... driverpath="CHROMEDRIVER PATH"...``` to be your Chromedriver path
